Running the server:
1)Open main.py
2)Change the IP address to your IPv4 address
3)Open command prompt and start main.py using Python

Running the Code in Android Studios:
1)Open the project and build project
2)open the java class "MainPage" and edit the url of the ip address to your IPv4 address
3)Build project and run the application in your android phone



Running the Arduino for Bicycle Lock System:
1)Open lockSystem.ino in Arduino
2)Compile and upload the code into your BLE device
3)Open the Serial monitor


Running the Arduino for Parking Lot Beacon:
1)Open parkingSystem.ino in Arduino
2)Compile and upload the code into your BLE device
3)Open the Serial monitor
